# EEG-Emotion-Detection
Emotion-based depression signals using Brain EEG Signals

This project leverages EEG Signals from the MODMA dataset in the resting and ERP states, and implements feature extraction and modelling on the EEG Signals to determine depressed states based on 3 emotional channels: happy, sad and fear.

Dataset Link: MODMA Dataset [https://modma.lzu.edu.cn/data/application/]
